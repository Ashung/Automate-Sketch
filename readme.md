# Automate Sketch

Make Sketch more efficient -- a lot of usefull plugins in one.

[中文说明](readme_zh.md)

## Installation

1. Download [master.zip](https://github.com/Ashung/Automate-Sketch/archive/master.zip), unzip it.
2. Double-click the "automate-sketch.sketchplugin".
